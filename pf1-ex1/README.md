# Backbase Training Exercise

## Portal Frontend - Module 1

### Exercise 1
note: This exercise is 1 of 5 in a series of exercises that follow on form each other and should not be used indepently
note: All exercies use the standard portal set-up for backbase trainings (Link to the archeype that we should use when developing on this course)
 
#### Description

To create your first widget that will become a todomvc example at the end of the series

#### Steps

 - Create the folder structure for a “Todo” widget 
 - Create your index.html
 - Add a “limit” preference
 - Create a file called server-catalog-widgets.xml and use YAPI to import it
